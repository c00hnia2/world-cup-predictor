# 🏆 World Cup Predictor

A full-stack web application built with Next.js and React that allows users to predict World Cup match outcomes, compete in global leaderboards, and create private leagues with friends.

## 📝 About The Project

World Cup Predictor is a web-based prediction game. Users can guess the exact scores of upcoming matches. The scoring system rewards accuracy and football knowledge:
- **1 Point:** Correctly predicting the match winner (or draw).
- **+3 Points:** Correctly predicting the exact match score (Total: 4 points).

### ✨ Key Features
- **Secure Authentication:** User registration with email verification and username/password login.
- **Global Leaderboard:** See how you rank against all users worldwide.
- **Private Leagues:** Create your own league and invite friends using a unique league code.
- **Real-time Match Updates:** View upcoming fixtures and submit predictions before kickoff.
